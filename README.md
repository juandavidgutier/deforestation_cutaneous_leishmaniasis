# Project Title

The relationship between municipal deforestation and the incidence of cutaneous leishmaniasis in Colombia: an ecological study using a causal machine learning framework

## Description

Code and dataset shared to reproduce the results of the paper The relationship between municipal deforestation and the incidence of cutaneous leishmaniasis in Colombia: an ecological study using a causal machine learning framework.
The file data.csv is the dataset used for the results of the causal machine learning framework presented in the manuscript.
To reproduce the results of the fine-tune, use the file R-score\_leishmaniasis.py.
To reproduce the results of the ATE and CATE use the file deforestation\_leishmaniasis.py.
